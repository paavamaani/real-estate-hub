### Prerequisites for local setup

Before cloning the repo, make sure you have the following installed:

- Node.js
- npm

### Installation

1. Clone the repository:
   git clone https://github.com/yourusername/real-estate-hub.git
   cd real-estate-hub

2. Install Node Modules
   npm install

3. Run the server
   npm start
